## 버그 수정 이력

### 1. `projectPath` 이중 슬래시
- 원인: `'/' + encoded` 후 `.replace(/-/g, '/')` 하면 선행 `-`가 `//` 생성
- 수정: `encoded.replace(/-/g, '/')` 로 단순화

### 2. `projectPath` 디코딩 오류
- 원인: Claude Code 인코딩이 `/`와 `_` 모두 `-`로 치환 → 역방향 복원 불가
  - 예: `private_2026/personal-coding-agent-monitor` → `private-2026-personal-coding-agent-monitor`
- 수정: `resolveSegments()` 로 파일시스템을 직접 탐색
  - 긴 매칭부터 시도 (최대 8파트), 구분자 순서: `-` → `_` → `.`
  - 매칭 실패 시 단일 파트로 폴백하며 계속 탐색

### 3. 세션 선택 후 이벤트 0개
- 원인: `useWebSocket`은 실시간 신규 이벤트만 수신 (watcher `ignoreInitial: true`)
- 수정: `handleSessionChange` 에서 REST API로 히스토리 선 로드, 이후 WebSocket 이벤트 append
  - id 기준 중복 제거로 동일 이벤트 두 번 표시 방지

### 4. 포트 3001 EADDRINUSE
- 원인: 테스트 중 실행한 임시 서버가 종료되지 않아 tsx watch가 크래시 후 재시작 못 함
- 해결: `kill <PID>` 후 루트에서 `npm run dev` 재실행
